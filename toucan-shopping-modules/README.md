# 犀鸟电商平台模块层
    这个目录下的所有代码可以适配微服务架构和标准版架构
    
    
##### 模块介绍

 | 模块名                                        |  说明                                                           | 
 | --------------------------------------------- | --------------------------------------------------------------- | 
 | adminAuth                                     | 权限管理                                                       |
 | area                                          | 地区                                                           |
 | category                                      | 商品分类                                                       |
 | colorTable                                    | 颜色表                                                         |
 | column                                        | 栏目                                                           |
 | common                                        | 公共模块(图片上传、日志、redis操作、异常处理等)                |
 | content                                        | 内容(轮播图)                                                  |
 | message                                        | 消息                                                          |
 | order                                          | 订单                                                          |
 | product                                        | 商品                                                          |
 | search                                         | 搜索                                                          |
 | seller                                         | 卖家                                                        |
 | skylark                                        | 云雀分布式锁                                                 |
 | stock                                          | 库存                                                         |
 | user                                           | 用户                                                         |
