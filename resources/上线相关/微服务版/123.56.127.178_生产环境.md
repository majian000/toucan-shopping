#### 介绍
    仅限在同一台机器上部署所有服务情况

#### 阿里云ECS linux 64位 CPU单核/内存8G

    jdk:1.8.0_181
    xshell linux
    
    
   
    
    ======================服务层===========================================
    
    消息服务
    启动:nohup java  -Xmx200m  -XX:+UseConcMarkSweepGC -XX:+UseParNewGC -XX:CMSInitiatingOccupancyFraction=77 -XX:+CMSParallelRemarkEnabled -XX:SoftRefLRUPolicyMSPerMB=0 -Djava.awt.headless=true -Dfile.encoding=UTF-8 -XX:ParallelGCThreads=4 -XX:SurvivorRatio=8 -XX:TargetSurvivorRatio=80 -XX:MaxTenuringThreshold=15 -verbosegc  -XX:+HeapDumpOnOutOfMemoryError -XX:+PrintGCDateStamps -XX:+PrintGCDetails  -jar toucan-shopping-message-1.0-SNAPSHOT.jar >/dev/null 2>&1 --spring.profiles.active=prod   --toucan.ip=123.56.127.178  --toucan.port=8103 --toucan.config.server.ip=8.140.187.184 --toucan.eureka.ip=8.140.187.184   &
    说明:设置最大可用堆内存

    用户服务

    说明:设置最大可用堆内存
